Deploy it yourself

Four steps to your own live copy.

  1. 01
    Fork this repo

    Click Fork on GitHub to get your own copy of the source code.

  2. 02
    Install Coolify

    Run the one-line install script on any VPS (Hetzner, DigitalOcean, Vultr…).

    curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash
  3. 03
    Create a new resource

    In Coolify: New Resource → Public Repository, paste your fork URL, choose the Dockerfile build pack.

  4. 04
    Add your domain & deploy

    Point your DNS at your server, set the domain in Coolify, and hit Deploy. SSL is automatic.
